Before running:
- install node version 12 or higher
- install openjdk version 11 or higher
- install chrome browser Version 93

For running open a terminal and type:
- npm run test

For running using jenkins server open terminal and type:
- npm run start-jenkins-server
- Go to http://localhost:8080/
- User and Pass: admin / admin
- Dashboard WebIo Build
- Build with Parameter -> test


Test descriptions are in cucumber files:
- /features/sendValidform.feature
- /features/inputValidation.feature
